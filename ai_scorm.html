Using this Tool on Squarespace

Squarespace's Code Block feature can sometimes restrict the functionality of file uploads for security reasons. If you find that the "Click to upload" or drag-and-drop area is not working, it's a limitation of the Squarespace platform, not an error in the tool.

The Recommended Solution:

  1. Host this tool externally: Save the entire code for this tool as an index.html file. Upload it to a free static hosting service like GitHub Pages or Netlify.
  2. Embed in Squarespace: Once hosted, you will get a public URL (e.g., https://your-username.github.io/scorm-tool/). Go back to your Squarespace page and add a Code Block. Instead of pasting this tool's code, embed it using an iframe: 

<iframe 
  src="YOUR_EXTERNAL_URL_HERE" 
  style="width: 100%; height: 900px; border: none;">
</iframe>

This method ensures full functionality by running the tool from a standard hosting environment that you can then display on your Squarespace site.
